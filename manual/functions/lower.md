# Function: lower

Transform a string to lower case characters.

## Syntax

```lower($string)```

## Arguments

|Index|Name|Type|Required|Default|Description|
|---|---|---|---|---|---|
|0|string|string|yes|-|Value to transform|

## Example

```
{$result = lower($string)}

{$result = $string|lower}
```
