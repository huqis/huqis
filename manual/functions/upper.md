# Function: upper

Transform a string to upper case characters.

## Syntax

```upper($string)```

## Arguments

|Index|Name|Type|Required|Default|Description|
|---|---|---|---|---|---|
|0|string|string|yes|-|Value to transform|

## Example

```
{$result = upper($string)}

{$result = $string|upper}
```
